TWITCH (API)

Introducción:

Para poder obtener datos de la API de Twitch es necesario, primero, tener una clave de acceso para aplicaciones en esta plataforma. Para obtener una clave debes visitar: https://dev.twitch.tv/console. Necesitarás hacer login o crear una cuenta nueva en caso de que no la tengas.

En el apartado "aplicaciones" puedes generar las credenciales para una nueva "aplicación", que es lo que vamos a usar para obtener dato. Ten en cuenta que NUNCA, bajo ningún concepto, debes hacer públicas estas claves.

Para Facilitar la comunicación entre la API y nuestro código, vamos a usar la librería TwitchAPI. Esta librería es bastante compleja en su última versión, por lo que recurrimos a una versión antigua, la V2.5.7.1. Para ello, debes abrir un terminal (en Windows es necesario usar PowerShell), y lanzar el comando siguiente: 

pip install twitchAPI==2.5.7.1

Una vez instalada la librería, podemos empezar a trabajar con ella. Aquí tenéis toda la documentación.

Cosas a tener en cuenta:

  1. Las peticiones a la API tienen un límite: En la documentación se establece cuantas peticiones puedes realizar en una ventana concreta de tiempo.
  2. Deberás añadir reguladores en el código para que la ejecución se pause para evitar superar estos límites.
  3. Recuerda: NUNCA publiques las claves de tu aplicación.

Codigo de ejemplo:

from twitchAPI.twitch import Twitch
import json

# TWITCH API VERSION 2.5.7.1
# pip install twitchAPI==2.5.7.1
# DOCS https://pytwitchapi.readthedocs.io/en/v2.5.7/modules/twitchAPI.twitch.html

CLIENT = ""
SECRET = ""

def twitch_example():
    twitch = Twitch(CLIENT, SECRET)
    result = twitch.get_streams(first=20, language="ca")
    print(result)
    
    # GUARDAMOS LA RESPUESTA EN UN DOCUMENTO JSON
    with open("output_file.json", 'w', encoding='utf-8') as f:
        json.dump(result, f, ensure_ascii=False, indent=4)
        
    # IMPRIMIMOS EL NOMBRE DE USUARIO Y LOS VIEWERS
    results = result["data"]
    for r in results:
        print(r["user_name"], r["viewer_count"])
    
twitch_example()

Ejercicios con la API

  1. Crea un código que pueda extraer todos los streamers en directo de Twitch en español que se encuentran ahora mismo en directo.
  2. Vuelca el contenido de cada consulta en un documento JSON distinto.
  3. Crea un dataframe con pandas con estos datos.
  4. Calcula el promedio de espectadores del momento.
  5. Exporta el dataset a un archivo .csv
  6. Analiza la distribución de espectadores dentro del dataset

Ejemplo completo y comentado realizado en clase

from twitchAPI.twitch import Twitch
import datetime
import pandas as pd
import time

now = datetime.datetime.now() # Agafem l'hora i dia en que iniciem la consulta

# Definim les credencials de la API
twitch = Twitch('app_id', 'clave_secreta')

# Definim una llista dels idiomes que volem capturar
idiomes = ["es","ca"]
cursor_dummy = None # Variable DUMMY per al cursor (paginació de resultats). Necessari per a la primera consutla de la API.

# En cada bucle d'idioma, els resultats de la API es van guardant en aquesta llista
llista_dataframes = []

# Funció recursiva principal
def crida(cursor, lang): # Passem dues variables: Cursor i idioma.
    streams = twitch.get_streams(first=100, language=lang, after=cursor) # Funció "get_streams()"
    dades = streams["data"] # Obrim el diccionari que obtenim de la API

    for dada in dades: # Per cada emissió en directe, agafem les dades que volem, i les posem en una variable.
        captured_at = now
        user_id = dada["user_id"]
        user_name = dada["user_name"]
        game_id = dada["game_id"]
        game_name = dada["game_name"]
        title = dada["title"]
        viewer_count = dada["viewer_count"]
        started_at = dada["started_at"]
        is_mature = dada["is_mature"]
        language = dada["language"]

        # Les dades de la variable, les volquem en un dataframe.
        df = pd.DataFrame({
            "captured_at": captured_at,
            "user_id": user_id,
            "user_name": user_name,
            "game_id": game_id,
            "game_name": game_name,
            "title": title,
            "viewer_count": viewer_count,
            "started_at": started_at,
            "is_mature":is_mature,
            "lang": language,
        }, index=[0])
        llista_dataframes.append(df)

    # Ara intententem agafar el cursor per a la següent pàgina
    try:
        cursor = streams["pagination"]["cursor"] # El cursor està al final del document de resposta de la API.

        # Si troba el cursor, imprimirá la següent linea, que ens diu també quin idioma estem treballant, i quants streams portem capturats.
        print(f"Fent una nova consulata. Idioma: {lang} - Total de Streams: {len(llista_dataframes)}")

        # Afegim un temps de pausa enrte consultes per evitar tocar el màxim de la API
        time.sleep(0.12)

        # Tornem a cridar la funció "crida" i li passem el nou cursor per a que continui
        crida(cursor, lang)

    # Afegim una excepció: En cas que no trobi el cursor, significarà que ha arribat a la última pàgina.
    except KeyError:
        print("Ultima Pagina")
        print(f"total streams capturats {len(llista_dataframes)}")

        # Això fa que la funció salti a la següent tasca.
        pass

# Aqui comença l'execució de l'script. El loop itera sobre la llista d'idiomes.
for lang in idiomes:

    # A cada bucle, netegem la llista de dataframes generats per l'anterior idioma.
    llista_dataframes.clear()
    # Iniciem l'script d'exrtacció
    crida(cursor_dummy, lang)

    # Quan hagi acabat de fer l'extracció, guardarà el dataset amb una extensió .csv i utilitzant l'idioma com a part del nom de l'arxiu.
    final_dataframe = pd.concat(llista_dataframes)
    final_dataframe.to_csv(f"export_{lang}.csv", index=False)
