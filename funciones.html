Funciones

Las funciones en Python, al igual que en todos los lenguajes de programación, son fragmentos de código que tienen un propósito concreto y realizan una tarea específica. Las funciones sirven para agrupar partes de un programa. Una función puede recibir argumentos que modifican sus parámetros. La funciones se puede reutilizar para ahorrar código.

Para iniciar una función, debes usar la siguiente estructura:

def nombre_funcion(opcional):
    contenido de la funcion
    return lo_que_sea

nombre_funcion(opcional)

Sintáxis:
  1. La partícula "def" identifica que se está creando una función
  2. Luego se determina el nombre de la función, que puede ser el que quieras
  3. Es obligatorio usar los paréntesis (), aunque pueden dejarse vacíos si no se está pasando ningún argumento a la función.
  4. Se cierra con dos puntos
  5. Todo lo que está indentado dentro de la función pertenece a la función
  6. La función puede retornar "return resultados"

Aquí tienes un ejemplo de una función. Esta se llama "checker", y tiene como objetivo verificar si la variable que está recibiendo es una cadena de texto que coincide con "joan". La función se encarga de verificar si se trata, o no, de joan. Dependiendo de if/else, la variable "resultado" obtiene un valor u otro. Luego este resultado se imprime.

lista = ["adria", "carla", "joan", "pere"]

def checker(nombre):
    if nombre == "joan":
        resultado = "Aqui está Joan"
    else:
        resultado = nombre+" No es joan"
    return resultado

for nombre in lista:
    resultado = checker(nombre)
    print(resultado)

>> adria No es joan
>> carla No es joan
>> Aqui está Joan
>> pere No es joan

Otra forma de expresar una función que haría, más o menos, lo mismo que la anterior, es la siguiente: 

lista = ["adria", "carla", "joan", "pere"]

def checker():
    for nombre in lista:
        if nombre == "joan":
            print("Aqui está Joan")
        else:
            print(nombre+" No es joan")
checker()

Tal como se ha dicho, las funciones admiten argumentos. Por ejemplo: hemos transformado la función anterior en una que admite un argumento, que es una variable donde se contiene el nombre que debe ser "checkeado". 

lista = ["adria", "carla", "joan", "pere"]
nombre_a_checkear = "carla"

def checker(nombre_a_checkear):
    for nombre in lista:
        if nombre == nombre_a_checkear:
            print("Aqui está "+nombre_a_checkear)
        else:
            print(nombre+" No es "+nombre_a_checkear)
checker(nombre_a_checkear)
