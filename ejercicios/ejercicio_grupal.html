Ejercicio Grupal

Este ejercicio consiste en crear artículo (con estilo periodístico) que presente (mediante texto y gráficos interactivos incrustados) una investigación, caso de estudio o análisis de un fenómeno basado en datos. Para ello, se deberá seleccionar una plataforma digital que disponga de una API abierta, y plantear un tema/caso. El artículo deberá incluir los siguientes apartados:

  1. Presentación del caso de estudio, incluyendo la motivación o interés que existe sobre la cuestión.
  2. Una explicación detallada del proceso de obtención de datos, incluyendo un link a repositorio de Github con codigo completo, y fragmentos de codigo incrustados en el artículo, donde se explique de forma clara cómo se han recolectado los datos.
  3. Una explicación de cómo se han procesado/combinado los conjuntos de datos.
  4. Una presentación del dataset resultante, que servirá para el análisis.
  5. Un texto con estilo periodístico que explique el fenómeno y presente gráficos interactivos para sustentar la información.
  6. Apartado de conclusiones o posibles futuras investigaciones.

Recursos

  1. Articulo de ejemplo
  2. Plantilla html para el articulo

Criterios de Evaluación

    Parte Código: (50%)
  1. Calidad general del código (claro, sencillo, funcional, ordenado).
  2. Calidad general de los comentarios (documentación).
  3. Aplicación de criterios de eficiencia (Tiempo/Recursos necesarios).
  4. Originalidad en la resolución de los problemas.

    5. Parte texto: (50%)
  5. Calidad general del texto.
  6. Calidad de los gráficos y visualizaciones realizados con Tableau.
  7. Capacidad para generar nuevas preguntas y plantear nuevos problemas.
  8. Originalidad, creatividad, capacidad narrativa, etc.
