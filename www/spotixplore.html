Objetivo del ejercicio

Objetivo: Crear un programa en Python que extraiga datos de la API de Spotify. Queremos generar un grafo que permita observar, estudiar y explorar el módo en cómo Spotify relaciona los artistas de la plataforma. Aquí un ejemplo:

Pasos a seguir:

  1. ¿La API de Spotify proporciona los datos que necesitamos para cumplir con el encargo?
  2. ¿La librería "Spotipy" tiene una función específica para solicitar los datos?
  3. ¿Se neceita algún tipos de autenticación de acceso a la API?
  4. ¿Hay algún tipo de limitación de cantidad de peticiones o datos?
  5. ¿Qué datos necesito para realizar la petición a la API?
  6. ¿En qué formato nos devuelve los datos la API de Spotify?
  7. ¿Qué datos adicionales tenemos disponibles?
  8. ¿Cómo puedo combinar estos datos para generar una visualización?

Ejemplo de implementación

import spotipy
from spotipy.oauth2 import SpotifyClientCredentials

## Reference

api_client_id = ""
api_client_secret = "" 

spotify = spotipy.Spotify(
client_credentials_manager=SpotifyClientCredentials(
	api_client_id,
	api_client_secret
	)
	)

results = spotify.artist_related_artists("5lkiCO9UQ8B23dZ1o0UV4m")
print(results)
